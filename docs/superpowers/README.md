# Superpowers Planning Archive

This directory stores implementation plans and design specs that were used to drive earlier development work. They are retained as historical engineering records, not as the primary current user manual.

For current behavior, prefer:

- [../iota book.md](../iota%20book.md) for the system-level technical guide.
- [../architecture.md](../architecture.md) for current module boundaries.
- [../code-call-chains.md](../code-call-chains.md) for current runtime paths.
- [../command.md](../command.md) for current CLI and TUI commands.
- [../desktop-mvp-acceptance.md](../desktop-mvp-acceptance.md) for current desktop acceptance scenarios.

Historical plans may include expected failures, obsolete intermediate code snippets, and commands that were valid at the time of planning. When a command appears in both this archive and the current docs, the current docs are authoritative.
