# Gefsi Experiment Archive

`gefsi/` contains dated experiment reports used to validate iota-sympantos behavior during development. These reports preserve the measurements, commands, versions, and observations from the time each experiment ran.

For current architecture and command syntax, use [../docs/iota book.md](../docs/iota%20book.md), [../docs/architecture.md](../docs/architecture.md), and [../docs/command.md](../docs/command.md). If a historical command here differs from the current docs, treat the report as evidence for that experiment date, not as the latest runbook.
